# InterioApp — Shopify Made-to-Measure Calculator

AI-powered quoting calculator for blinds, curtains, shutters & wallpaper. Built for Shopify merchants worldwide.

## Overview
InterioApp helps curtain and blind retailers sell made-to-measure products online with accurate, region-specific quoting powered by AI.

## Target Markets
- 🇺🇸 United States
- 🇬🇧 United Kingdom
- 🇳🇿🇦🇺 New Zealand & Australia
- 🇨🇦 Canada
- 🇪🇺 European Union

## Tech Stack
- **Framework:** Remix + React
- **UI:** Shopify Polaris
- **Hosting:** Heroku
- **Database:** Supabase (PostgreSQL)
- **CI/CD:** GitHub Actions
- **AI:** Measurement validation, pricing, recommendations
